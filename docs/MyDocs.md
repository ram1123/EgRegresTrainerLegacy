# Setup

- Setup virtual environment

    ```bash
    python -m venv .venv
    ```

- Activate the environment and update pip

    ```bash
    source .venv/bin/activate
    pip install --upgrade pip
    pip install -r reuirements.txt
    ```

- Content of `requirements.txt`

    ```bash
    sphinx
    livereload
    myst-parser
    ```
